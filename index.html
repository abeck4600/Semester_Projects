Pokémon - Factory Pattern

In 3D Programming 2 Course, The assignment includes a Pokemon battle structure, a free-roaming player movement scene with random encounters, and detailed script documentation. The Factory Pattern streamlines object creation, focusing on code organization.

The PokémonFactory class creates Pokémon objects, obtaining data from a Pokedex.csv file. To use these objects in Unity, the PokemonComponent class, inheriting from MonoBehaviour, is employed. All Pokémon variables are made public for easy inspection. A script interacts with these Pokémon objects. Wild Pokémon encounters are initiated within the movement script when the player enters grass, activating the BattleCamera and resetting the player's position.

Motorstrying PF

In Hardware and Robotics 1 course,
I designed, constructed, and manufactured a Printed Circuit Board (PCB) using the Eagle software, for motor control, specifically tailored to control the direction and speed of a single DC motor. The heart og this project is the H-bridge that was configured using MOSFETs(NPN and PNP) with the addition of flyback diodes for robust performance. The direction of current flow through the H-bridge was controlled using a comparator that inverts an input signal, ensuering a consistent output of either (high/low) or (low/high) and a strategically placed low-side switch between the H-bridge and motor's power supply ground (GND). This project has allowed me to create a versatile motor control circuit to my specifications.

Model View Controller pattern in PHP

In Webprogramming 3 course,
My website is built using the MVC design pattern, implemented in PHP with MariaDB as the database. The site features a menu panel in the top-left corner for easy navigation.
One of the security measures in place is that only registered users can upload images to the database. When creating a user profile, a RegEx validation enforces that the email address must be in lowercase and include an '@' symbol.
Images uploaded to the site are stored as Base64-encoded data. This approach is chosen to manage the storage efficiently and prevent the database from getting cluttered with large image files.
To prevent XSS (Cross-Site Scripting) attacks, input data is sanitized using FILTER_SANITIZE_FULL_SPECIAL_CHARS.
Protection against SQL injection is achieved by using bindParam, which prevents malicious SQL code from being entered into text fields.
User passwords are securely protected through hashing. This means that the original passwords cannot be retrieved from the hash values. In the database, passwords are stored as hashed text strings.

4. Semester

Knas med Kredsløb - VR

In Learning Technology 2,
I created a VR 3D game using Unity, named "Knas med Kredsløb - teknologiforståelse i grundskolen."
This game was designed as a digital learning tool. VR technology enabled me to leverage relevant learning theories, making it more accessible and engaging for learners. Within the game, I simulated electrical circuits on a breadboard, something that is not feasible in reality. This unique feature allowed students to gain a better understanding of technical subjects and reduced the risk of damaging actual components.
The technology underwent testing with students between the ages of 10 and 14 at Teknologiskolen.

Hjertesti - AR

In Virtual and Augmented Reality 1,
I created a project called “Den Digitale Hjertesti.”
It was a digital hjertesti, enabling individuals without access to a physical hjertesti location to participate digitally using their phones.
When the walking/running person symbol intersected with a heart symbol along the route, a video of me performing exercises would appear.
A hjertesti is a designated exercise route where signs displaying the Heart Association's logo are positioned along the path. By digitizing it, everyone can participate with the aim of improving people's health.
The current digital route is located in Munke Mose, Odense C
This project was made possible by using Unity 3D, Blender, and Bing Maps 3D world data to Incorporate depth, information, and more into the application.
